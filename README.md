# Person-Classifier-AI 🚀

An end-to-end machine learning application that classifies individuals into 10 distinct categories based on 28 biometric and behavioral features.

## 🛠 Tech Stack
- **AI:** Scikit-Learn / TensorFlow
- **Backend:** FastAPI (Python)
- **Frontend:** React.js + Tailwind CSS
- **Containerization:** Docker
